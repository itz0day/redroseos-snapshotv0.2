**🌹 RedROSE OS - Snapshot v0.2**

The second snapshot of RedROSE OS focuses on deep privacy, network anonymity, and pure performance. This version introduces custom security modules and a completely revamped visual identity.

### 🛠 System Specifications
<p><b>Kernel:</b> 6.22.x-arch1-1 (Latest Stable)</p>
<p><b>Base:</b> Arch Linux</p>
<p><b>DE:</b> XFCE 4.18 (High-Performance Tweak)</p>
<p><b>Shell:</b> Zsh (Custom RedROSE-Zsh integration)</p>
<p><b>Terminal:</b> RedROSE Terminal (Newly Integrated)</p>

---

### 🚀 New Features in v0.2

* **⚡ Copy-to-RAM (Live Mode):** Load the entire system into memory for blazing-fast performance and disk-less operation.
* **👻 GhostZone (Privacy Mode):** A dedicated identity masking suite.
    * `gz-on`: Enables stealth mode and identity masking.
    * `gz-off`: Reverts to standard operation.
* **🌐 Tor Global Routing:** System-wide network traffic is now routed through the Tor network for maximum anonymity.
* **🎨 Visual Overhaul:** Completely new RedROSE branding, including a custom logo and wallpaper set.
